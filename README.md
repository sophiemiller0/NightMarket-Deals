# NightMarket Deals
一个用于抓取与汇总本地夜市折扣信息的个人项目。

使用方式（本地脚本）：
- 准备好 Python 3.11+ 环境
- 执行 `python scripts/export_csv.py` 生成 `data/deals.csv`
- 目前数据源为脚本内置的 `mock_source()`，后续会逐步接入更多来源

后续计划：
- 增加简单关键词过滤与价格区间筛选
- 引入基于标题相似度的去重策略
- 增加 `tests/` 下的单元测试样例

目标：
- 抓取来自若干源（手动导入、模拟API）的折扣信息
- 提供本地CSV导出与简单查询脚本
- 逐步演进（如添加打分、去重、订阅等）
